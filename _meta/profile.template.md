# Personal profile template

> **How to use:** copy this file to `personal/profile.md` (the `personal/` directory is
> gitignored — see `.gitignore` — so nothing here is ever committed or published). Fill in
> the facts that resolve the conditional rows in `synthesis/ranked-actions.md` and inform
> your Tractability scores. See `synthesis/personalize.md` for the full method.
>
> Replace every `<...>` placeholder. Delete lines that don't apply. Date the snapshot — a
> profile is a point-in-time thing; revisit it when your status changes.
