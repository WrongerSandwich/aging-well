# Lever × System matrix

The many-to-many map. Build this AFTER you have a few lever sessions done. It's the
structural defense against double-counting: a lever researched once shows up here
against every system it protects, instead of being "rediscovered" per organ.

Fill cells with effect strength: **●** strong · **◐** moderate · **○** minor · blank = none.

## Reading the matrix
- **Dense rows** (a lever with many ●) = highest-leverage interventions. These should
  dominate `ranked-actions.md`. Expect exercise, sleep, substances, metabolic to be dense.
- **Sparse/empty columns** = a system with no strong lever pointing at it = a coverage
  gap. Investigate before declaring the project done.
- A claim counted once as a dense row is NOT five separate findings. The matrix makes
  that visible so the ranking doesn't inflate.
